Deriver tout les objets de la classe Object

Declarer les references : xxxRef pour xxx*

Syntaxe :
- premiere lettre en minuscule, les autres débuts de mot commencant par une majuscule
- variable membre d'une classe commence par m_
- variable globale commence par g_
- pas d'autre "_" dans le nommage des variables et/ou fonctions
- tout element d'enumeration doit commencer par un 'e' suivi de terme parlant avec la premiere lettre en majuscule

Toujours creer un constructeur par defaut
